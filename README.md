# bigdata-subway-station-analysis

# 신규 지하철역 필요 지역 분석 시스템 설계

## 1. 프로젝트 개요 및 문제 정의
현재 수도권 대중교통망은 촘촘하게 구축되어 있지만, 명지대학교처럼 지역별 실제 유동인구나 밀집도에 비해 지하철 접근성이 떨어지는 '교통 소외지역'이 존재함. 본 프로젝트는 단순한 지리적 접근성을 넘어, 실제 생활 인구와 기존 교통수단(버스, 지하철)의 혼잡도 데이터를 종합적으로 분석해 새로운 지하철역 신설의 타당성이 가장 높은 지역을 도출하는 것을 목표로 함. 

## 2. 사용 데이터 (Data Sources)
아래의 공공데이터를 사용하여 분석함.
* 서울교통공사_지하철혼잡도정보
* 한국도로공사_혼잡빈도
* 한국관광공사_관광빅데이터 정보서비스_GW
* 서울특별시_지하철 호선별 역별 시간대별 승하차 인원 정보
* 서울특별시_버스노선별 정류장별 승하차 인원 정보
* 서울특별시_자치구단위 서울생활인구 일별 집계표
